趣谈Linux操作系统 筆記
linux
发布日期:   2019-04-06
阅读次数:  

此心得筆記為閱讀極客時間-趣谈Linux操作系统

爬过这六个陡坡，你就能对Linux了如指掌

1.抛弃旧的思维习惯，熟练使用 Linux 命令行

《 鸟哥的 Linux 私房菜 》《 Linux 系统管理技术手册 》

2.通过系统调用或者 glibc，学会自己进行程序设计

《Unix/Linux编程实践教程》 ->《UNIX 环境高级编程》

3.了解 Linux 内核机制，反复研习重点突破

《 深入理解 LINUX 内核 》
《 庖丁解牛Linux内核分析》

4.阅读 Linux 内核代码，聚焦核心逻辑和场景

《 LINUX 内核源代码情景分析 》

5.实验定制化 Linux 组件，已经没人能阻挡你成为内核开发工程师了

从只看内核代码，到上手修改内核代码
制化自己的模块。例如，你可以自己实现一个设备驱动程序，实现一个自己的系统调用，或者实现一个自己的文件系统等

6.最后一坡：面向真实场景的开发，实践没有终点

你可以把Linux内核当成一家软件外包公司的老板

User -> Keyboard & Mouse(Input Device) -> Input Driver -> Interrupt -> Interrupt Service Routines(ISR) -> Binary Program -> File Management Subsystem -> System Call Subsystem -> Resource Allocation -> Process Management Subsystem -> Memory Management Subsystem -> Execute Program(Process) -> Output Driver -> Output Device(Display)

User -> 客戶需求 -> 客戶對接員 -> 停止原有服務了解客戶需求 -> 處理需求的方法 -> 需求立項 -> 项目计划书 -> 保存文件檔案庫 -> 辦事大廳(資源分配) -> 成立項目組(開發人員、項目經理、項目管理流程) -> 項目管理系統 -> 會議室管理系統 -> 執行项目计划 -> 交付人員 -> 展示需求


每家公司都有自己的黑话

常用的 command line 指令

用戶與密碼

passwd 修改密碼

useradd 創建用戶

useradd -g 群組設定

mac 沒有 useradd 指令

1.System Preferences -> Users & Groups 設定

2.使用 dscl command

显示所有 user 对应的 group

$ sudo dscl . -list /groups GroupMembership

添加 user 到 group

$ sudo dscl . -append /Groups/groupname GroupMembership username

从 group 中删除 user

$ sudo dscl . -delete /Groups/groupname GroupMembership username

查看 User、家路徑、與

# cat /etc/passwd
root:x:0:0:root:/root:/bin/bash
......
cliu8:x:1000:1000::/home/cliu8:/bin/bash

查看 Group

# cat /etc/group
root:x:0:
......
cliu8:x:1000:
